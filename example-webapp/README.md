# example-webapp

Angular 19 login system with PrimeNG + Tailwind CSS.

## Versions

- Node: 18.20.8
- Angular CLI: 19.2.25
- Angular: 19.2.21
- PrimeNG: 19.1.4
- Tailwind CSS: 3.x

## Setup

```bash
npm install
ng serve
```

Open http://localhost:4000

## API

Requires example-webapi running at http://localhost:8080/example-api

## Pages

- `/login` — เข้าสู่ระบบ
- `/register` — สมัครสมาชิก
- `/welcome` — หน้าหลัง login (ต้อง auth)
